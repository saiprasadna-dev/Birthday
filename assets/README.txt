Put your photos and music here.

PHOTOS
  Drop images like photo1.jpg, photo2.jpg, ...
  Then open js/config.js and set each memory's "img" to e.g. "assets/photo1.jpg".
  Recommended: landscape-ish images (4:3 works best), under ~1 MB each.

MUSIC
  Drop an mp3 named music.mp3 here (or any name — update "musicFile" in js/config.js).
  Use a song you have the rights to use.

Anything left blank falls back to a tasteful placeholder / stays silent —
nothing will break if a file is missing.
